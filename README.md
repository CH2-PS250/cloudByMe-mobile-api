# REST API Cuaca

REST API prakiraan cuaca, suhu udara, kelembapan udara, kecepatan angin, dan arah angin untuk kota-kota besar di **34 provinsi** di Indonesia.

Sumber: [Data Terbuka BMKG](https://data.bmkg.go.id/).

## Cuaca

### Provinsi

**Endpoint:**

`/{provinsi}`

`/jawa-barat`

### Kota

**Endpoint:**

`/{provinsi}/{kota}`

`/jawa-barat/bandung`
